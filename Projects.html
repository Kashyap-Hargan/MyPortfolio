Project Title: E Commerce Website
Description: This project is a fully functional e-commerce website designed for online watch retail, developed as part of my academic work. The backend of the website is built using Python with the Django framework for robust server-side functionality, ensuring seamless operations such as user authentication, order management, and product handling.
The frontend utilizes HTML, CSS, and jQuery, providing an interactive and user-friendly interface for customers to browse products, add items to their cart, and complete purchases effortlessly.
The website's data management is handled through MySQL, where all customer details, product information, and order records are securely stored and retrieved efficiently.

Key Features: Dynamic Product Display Watches categorized for easy browsing. CRUD Operations: Admin can create, read, update, and delete product details. User Authentication: Secure login and registration for customers. Shopping Cart and Checkout: Smooth purchase experience for users. Payment Gateway Integration: Facilitates online payments. Responsive Design: Ensures usability across devices.
